# mon-jeux-aethric-

RPG original (œuvre originale) développé sous **Godot 4.7 / GDScript**, destiné en priorité
au **Web mobile (iPhone 13, Safari)**.

Le jeu s'inspire de catégories générales de systèmes de RPG (exploration, combat tour par tour,
classes évolutives, équipement, loot, monstres, quêtes, donjons) avec **son propre univers, ses
propres noms, ses propres données et sa propre identité visuelle**.
